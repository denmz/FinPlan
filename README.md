# FinPlan
FinPlan is a modern, mobile-friendly web application designed to help users take control of their personal finances. The app provides an intuitive interface for tracking income, expenses, savings goals, and accessing financial education tools — all in one clean and responsive UI.

Features
🔐 Login System
Simple credential validation
Demo login included
Error handling and UI feedback
💰 Income & Expense Tracking
Add, edit, and organize financial entries
Clean category-based layout
Visual indicators and color-coded tags
🎯 Savings Goals
Create and monitor savings goals
Progress bar with dynamic updates
Detailed modals for each goal
📚 Financial Education Hub
Tiles with curated topics
Modal pop-ups for in-depth learning
User-friendly layout for easy browsing
🎨 Modern, Mobile-First UI
Custom CSS theme with navy, lime green, gold, and teal palette
Optimized hero banner, buttons, and navigation
Sliding side-menu and bottom navigation bar
Smooth transitions and responsive components
🛠️ Tech Stack
HTML5
CSS3 (custom theme + responsive design)
JavaScript (DOM handling + UI logic)
Google Fonts (Nunito & Baloo 2)
